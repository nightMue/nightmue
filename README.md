# nightMue Photography

A modern, high-performance photography portfolio and client experience platform built with Flutter Web.

Created and maintained by **Nick Mueller** under the **nightMue** brand, specializing in motorsports photography, automotive content, and visual storytelling.

---

## Overview

The NightMue Photography website serves as:

- Professional photography portfolio
- Client acquisition platform
- Image showcase gallery
- Client delivery portal
- Contact system
- Brand hub for social media and photography services

The site is designed to deliver a premium, cinematic experience while maintaining fast load times and responsive performance across desktop, tablet, and mobile devices.

---

## Features

### Portfolio Gallery

- High-resolution image display
- Category-based filtering
- Full-screen image viewer
- Image metadata support
  - Title
  - Camera
  - Lens
  - Year
  - Location
  - Description

### Client Portal

- Password-protected galleries

### Contact & Booking
